## 0. 들어가기 전

### 그룹 챗봇이란?

그룹 챗봇은 **그룹 채팅방(팀채팅/일반채팅)에서 작동하는 챗봇**입니다.

- 현재 **팀채팅**에만 지원됩니다.
- 나와의 채팅, 오픈채팅은 현재 지원하지 않습니다.
- 채널 1:1 채팅방 챗봇과는 별도이며, 하나의 채널에 그룹 챗봇과 1:1 챗봇 둘 다 연결 가능합니다.

### 채널 구분

| 채널 종류 | 용도 |
|-----------|------|
| **운영 채널** | 실 서비스에 연결하여 노출할 채널 |
| **개발 채널** | 챗봇 개발 목적으로 연결하는 채널 |

> 운영 채널과 개발 채널은 **서로 다른 채널**을 사용해야 합니다.
> 개발 목적만을 위한 개발 채널을 별도로 생성해 연결해 주세요.

### 사전 준비 사항

1. **챗봇 시나리오 작성** - 카카오에 전달해야 합니다.

   만렙개미 봇 시나리오 예시:
   - 챗봇 이름: `만렙개미`
   - 챗봇 주요 기능: 가상 주식 투자 게임
     - 실시간 주식 시세 조회
     - 가상 머니로 매수/매도
     - 예측 게임 (보물상자, 등락예측, 시장예측)
     - 랭킹 및 PvP 배틀
   - 대표 시나리오:
     - 시세 조회 알림
     - 투자 게임 진행
     - 랭킹 경쟁

   > 챗봇 응답만으로 사용자에게 충분한 정보를 제공해야 합니다.

2. **사업자 여부 확인** - 사업자가 아닌 경우 일부 API 사용에 제한이 있습니다.
   - 카카오 디벨로퍼스 앱 연결은 사업자번호가 필수
   - 카카오싱크/카카오로그인 관련 API 제한
   - 채널 프로필에 경고 문구 표시 (미표시하려면 별도 검토 필요)

### 프로필 가이드

1. **일반적이고 보편적인 이름은 지양** - 카테고리를 대표하는 보편적 명사 + "봇" 형태 사용 불가 (예: 게임봇, 주식봇)
2. **프로필 이미지 내 글자 사용 지양** - 이미지 크기로 인해 글자 가독성이 낮음
3. **리스크가 발생할 수 있는 키워드/이미지 지양** - 캡처되어 이슈화될 수 있는 케이스는 진행 불가

### 고객센터 안내

채널 홈에 고객센터 경로(이메일, 전화번호 등 CS 인입 루트)를 안내해야 합니다. 채널 1:1 채팅방을 활용해도 좋습니다.

## 1. 채널 연결하기

### 1-1. 카카오 비즈니스 가입

1. [카카오 비즈니스](https://business.kakao.com) 접속
2. 카카오톡에서 사용하는 카카오 계정으로 가입/로그인

### 1-2. 카카오톡 채널 만들기

운영 중인 채널이 있다면 그대로 사용할 수 있습니다.

채널명 작성 시 유의사항:
- 어떤 기능/서비스인지 직관적으로 이해 가능해야 함
- 타이핑하기 쉽고 간결해야 함 (대소문자, 띄어쓰기 X)
- 프로필 이미지 내 텍스트 사용 지양

> 채널명 변경 문의: 카카오 비즈니스 고객센터

### 1-3. 비즈니스 채널 인증

생성한 채널을 비즈니스 채널로 인증받습니다.

- **사업자인 경우**: 사업자 정보 입력
- **개인인 경우**: 사업자 정보 입력 단계에서 "건너뛰기" 클릭
  - 단, 디벨로퍼스 > 카카오로그인을 사용하려면 사업자만 가능

### 1-4. 개발용 채널 만들기

챗봇 개발/테스트에 사용할 채널을 **별도로** 만들어야 합니다.

**사업자인 경우:**
1. 카카오 비즈니스 > 내 채널 > "개발자용 채널 만들기" 클릭
2. 또는 https://business.kakao.com/profiles 에서 개발자용 채널 만들기

> 개발자용 채널과 연결된 봇은 모든 사용자에게 노출되지 않으며, 테스트 목적의 팀채팅방 1개에 한해 사용할 수 있습니다.

**개인인 경우:**
- 일반 채널 2개를 만들어 1개는 서비스용, 1개는 개발용으로 사용

### 1-5. 채널 공개 설정

채널 공개를 **On**으로 설정해야 카카오톡에서 노출됩니다.

### 1-6. 카카오싱크(로그인) 사용 시 (선택)

카카오 디벨로퍼스에서 앱을 등록하고 채널과 연결합니다. (사업자만 가능)

1. 카카오 디벨로퍼스 > 내 애플리케이션 > 애플리케이션 추가하기
2. 카카오 디벨로퍼스 > 내 애플리케이션 > 앱 설정 > 앱 권한 신청
3. 카카오톡 채널 연결 관리

## 2. 권한 받기

### 2-1. 챗봇 관리자센터 로그인

[챗봇 관리자센터](https://chatbot.kakao.com) 접속 후 카카오톡을 사용하는 계정으로 로그인합니다.

### 2-2. 관리자 카카오 계정 전달

챗봇을 제작/운영하기 위해 권한을 받을 관리자의 **카카오 계정**을 카카오에 전달합니다.

준비 사항:
1. [카카오 비즈니스](https://business.kakao.com) 가입
2. [챗봇 관리자센터](https://chatbot.kakao.com) 로그인
3. 카카오톡 로그인 (마스터 권한을 받을 1명만 진행)

> 권한을 받으면 챗봇 관리자센터에 **"그룹 챗봇"** 탭이 생기며, "그룹 챗봇 만들기"를 통해 다음 단계를 진행할 수 있습니다.

## 3. 사전 세팅

### 3-1. 채널과 챗봇 연결

챗봇 관리자센터 > 설정 > 챗봇 관리 > 카카오톡 채널 연결

1. **개발 채널** 선택하여 연결
2. 개발 채널로 연결하면 배포를 하지 않아도 수정사항을 바로 확인 가능

> 개발 채널과 운영 채널은 각각 서로 다른 채널을 연결하세요.

### 3-2. 챗봇 한줄 설명 작성

챗봇 관리자센터 > 봇 설명

- 팀채팅 챗봇 목록에 노출되는 설명입니다.
- 말줄임 없으려면 **14자 이내** 작성 권장
- 예시: `가상 주식 투자 게임`

### 3-3. 챗봇 도움말 작성

챗봇 관리자센터 > 시나리오 > 도움말 블록

사용자가 "도움말" 버튼을 클릭했을 때 표시되는 내용을 작성합니다.

만렙개미 도움말 예시:
```
만렙개미 - 가상 주식 투자 게임

주요 명령어:
/시작 - 게임 시작 (1,000만원 지급)
/출석 - 일일 출석 보상
/시세 [종목명] - 주식 시세 조회
/매수 [종목명] [수량] - 주식 매수
/매도 [종목명] [수량] - 주식 매도
/포트폴리오 - 전체 자산 현황
/랭킹 - 수익률 TOP 10
/도움말 - 전체 명령어 안내
```

### 3-4. 웰컴 메시지 (봇 입장 블록) 설정

챗봇 관리자센터 > 시나리오 > 봇 입장 블록

채팅방에 챗봇을 추가하면 발송되는 메시지입니다.

- 채팅창 절반 이하로 영역이 차지하도록 **내용을 길지 않게** 작성
- "@챗봇 멘션하기" 버튼 셋팅 (입력창에 @멘션 자동 입력)

### 3-5. 대표 명령어 설정

추천하고 싶은 대표 명령어를 설정합니다.

만렙개미 대표 명령어 예시:

| 명령어 | 설명 |
|--------|------|
| /시세 | 종목명을 함께 입력하세요 (예: /시세 삼성전자) |
| /매수 | 종목명과 수량을 입력하세요 (예: /매수 삼성전자 10) |
| /포트폴리오 | 보유 자산 현황 확인 |
| /랭킹 | 수익률 순위 확인 |

> 명령어가 화면에 반영되기까지 채팅방 진입 후 최대 1시간이 소요될 수 있습니다.
> 바로 반영되려면 챗봇을 제거하고 다시 추가해보세요.

## 4. 챗봇 개발 및 테스트

### 4-1. 개발 가이드

- [그룹 챗봇 가이드](https://kakaobusiness.gitbook.io/main/tool/chatbot) - 팀채팅에 제공되는 기능 가이드
- [기본 가이드](https://kakaobusiness.gitbook.io/main/tool/chatbot) - 공통 가이드 (팀채팅에 제공되지 않는 기능도 포함)

### 4-2. 스킬로 전달되는 주요 데이터

그룹 챗봇에서는 `userRequest` 내에 추가 데이터가 전달됩니다:

| 필드 | 설명 |
|------|------|
| `userRequest.block.id` | 호출된 블록 ID |
| `userRequest.block.name` | 호출된 블록 이름 |
| `userRequest.user.properties.botUserKey` | 봇 기준 유저 비식별키 (채널에 연결된 봇 변경 시 변경됨) |
| `userRequest.user.properties.plusfriendUserKey` | 채널 기준 유저 비식별키 (채널 변경 시 변경됨) |
| `userRequest.user.properties.appUserId` | 카카오로그인 가입 유저인 경우 발급된 키 |
| `userRequest.user.properties.appUserStatus` | 카카오로그인 가입/연결 상태 (`registered`) |
| `userRequest.chat.properties.botGroupKey` | **채팅방을 구분하는 비식별키** (그룹 챗봇 전용) |
| `userRequest.utterance` | 사용자 발화 |
| `userRequest.params` | 파라미터 |
| `userRequest.lang` | 언어 |
| `userRequest.timezone` | 시간대 |

> `botGroupKey`는 그룹 챗봇에서 채팅방을 구분하는 핵심 키입니다.
> 만렙개미에서는 이 키를 활용하여 채팅방별 랭킹/배틀 등을 구현할 수 있습니다.

### 4-3. 주요 개념

- **시나리오**: 사용자가 경험하는 챗봇 기능의 단위 (예: 시세조회, 거래, 예측게임)
- **블록**: 사용자 의도와 연결되는 가장 작은 단위. 하나의 시나리오는 여러 블록으로 구성 가능
- **팀채팅 전용 블록**: 봇 입장 블록, 도움말 블록

### 4-4. 사용자 발화와 챗봇 응답

사용자 발화와 매칭되는 블록을 추출 후 챗봇 응답을 출력합니다.

만렙개미에서는 `/skill` 엔드포인트에서 모든 명령어를 처리합니다:

```
사용자: @만렙개미 /시세 삼성전자
  -> utterance: "/시세 삼성전자"
  -> CommandHandler에서 명령어 파싱 및 처리
  -> KakaoResponse로 응답 생성
```

### 4-5. 엔티티와 패턴 발화 활용

- **엔티티**: 봇이 이해하는 용어 사전 (동의어 등록 가능)
  - 나의 엔티티: 관리자가 등록한 엔티티
  - 시스템 엔티티: 챗봇 관리자센터에서 공통으로 지원하는 엔티티
- **패턴 발화**: 사용자 발화 패턴을 등록하고 엔티티를 연결

### 4-6. 그룹 챗봇에서의 만렙개미 스킬 서버 설정

현재 만렙개미의 스킬 서버 엔드포인트(`/skill`)는 1:1 챗봇과 동일하게 사용할 수 있습니다.

그룹 챗봇에서 추가로 활용할 수 있는 데이터:
- `botGroupKey`: 채팅방별로 다른 게임 세션/랭킹 운영 가능
- `@멘션`: 그룹 채팅에서는 `@채널명 /명령어` 형태로 호출

## 5. 심사 기준 / 유의사항

### 심사 기준

- 챗봇 응답만으로 사용자에게 **충분한 정보를 제공**해야 합니다.
- 자사 웹이나 앱으로 랜딩하려는 경우, 챗봇 응답만으로 충분한 정보를 제공하면서 랜딩해야 합니다.
- 프로필 이미지와 이름이 캡처되어 이슈화할 수 있는 케이스는 진행이 어렵습니다.

### 유의사항

1. **운영 채널과 개발 채널을 반드시 분리**하세요.
2. 챗봇 서비스 오픈 후 스펙 업데이트 시에도 개발 채널에서 먼저 테스트합니다.
3. 프로필 이름/이미지 관련 가이드를 준수하세요.
4. 채널 홈에 고객센터 연락처를 반드시 안내하세요.
